# Auton Health — Landing

Landing page oficial da Auton Health — a 1ª IA de causa raiz da saúde integrativa.

## Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- Lucide React

## Dev local
```bash
npm install
npm run dev
```

Abre em http://localhost:3000

## Deploy
Deploy automático na Vercel a partir da branch `main`.
